# 🐶 Cachorros Web

Proyecto Astro sencillo convertido desde un HTML estático a una experiencia modular en español, con secciones independientes y recursos locales en `public`.

## Estructura

- `src/components/sections`: una sección por componente.
- `src/layouts`: layout único en español.
- `public`: recursos optimizados en la raíz.
- `public/originals`: copias originales preservadas.

## Desarrollo

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
